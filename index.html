Tentang Saya

Saya adalah lulusan SMK dengan spesialisasi di bidang Rekayasa Perangkat Lunak (RPL). Selama masa pendidikan, saya mempelajari berbagai aspek pengembangan perangkat lunak, termasuk dasar-dasar pemrograman, analisis sistem, serta design web. Saya memiliki ketertarikan kuat dalam teknologi dan terus belajar untuk memperluas kemampuan saya, baik di bidang software maupun hardware. Saat menjalani Praktik Kerja Lapangan (PKL), saya mendapatkan pengalaman langsung dalam bidang hardware, seperti perakitan komputer, troubleshooting perangkat keras, serta instalasi sistem operasi dan software pendukung. Pengalaman ini memperkuat pemahaman saya tentang infrastruktur teknologi dan membuat saya lebih fleksibel dalam menghadapi berbagai kebutuhan di dunia kerja.
